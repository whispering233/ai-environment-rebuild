# Claude Code Quick — 快速安装 Claude Code

## 适用场景

已有完整开发环境（Node.js 18+ 已安装），只需安装 Claude Code。

## 前提条件

| 条件 | 说明 |
|------|------|
| Node.js | 版本 18+ |
| npm | 随 Node.js 一同安装 |
| Anthropic API Key | 从 [console.anthropic.com](https://console.anthropic.com) 获取 |
| 操作系统 | macOS / Linux / WSL2 |

| 步骤 | 内容 | 引用知识文件 |

| [Step 2: 安装 Claude Code](./step-02-install-claude-code.md) | 安装 Claude Code 并配置 API Key | `agents/claude-code/README.md` |

1. AI 依次阅读各步骤文件
